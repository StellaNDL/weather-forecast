# 🌦️ Weather Forecast App

A responsive weather application that allows users to search for any city and view real-time weather data.

## 🔗 Live Demo
https://weather-forecast-application18.netlify.app/

## 🚀 Features
- Search for any city
- Displays real-time weather data
- Responsive design for mobile and desktop
- Dynamic UI updates based on API data

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Weather API
